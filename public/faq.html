What kind of encryption does ovpn.io use?

OpenVPN provides a number of ciphers to choose from. We choose the most secure ones for ovpn.io:

  • 4096 bit private keys and DH parameters
  • MitM & DoS protection
  • Control Channel
    • TLS version 1.2
    • TLS-DHE-RSA-WITH-AES-256-GCM-SHA384 TLS cipher
  • Data Channel
    • SHA512 digest
    • AES-256-CBC cipher


Where is ovpn.io hosted?

ovpn.io is hosted in France, inside an OVH datacenter

We rent a dedicated server, therefor no hypervisors are involved. We are not vulnerable to any attacks which concern virtualization because nothing is virtualized. Resources are not shared. The whole machine is solely dedicated for ovpn.io

The server has the following specs:

  • Core i5-750 (4 cores @ 2.67GHz)
  • 16GB of DDR3 RAM
  • 100Mbps bandwidth

This is a serviceable machine. However the location and bandwidth are not ideal. We are aware of the issue and we will try to migrate the server to a more privacy-friendly country, on a datecenter with more bandwidth (while keeping the same price)
